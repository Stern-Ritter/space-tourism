# Проект: лендинг NASA.

### Описание проекта
Перед вами лендинг для космических туристов.
Он содержит описание предложения по космо-туризму:
* возможные маршруты,
* информацию о компании и способы оплаты.
Лендинг сделан с помощью дизайн-системы NASA.
